\subsection{Example Mathematical Expression}
\label{sec:math-example}

Here's an example of how to use the custom math commands defined in \texttt{customcommands.sty}:

The derivative of a function $f: \R \to \R$ can be written as:
\begin{equation}
    \deriv{f(x)}{x} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
    \label{eq:derivative}
\end{equation}

For a vector $\vect{v} \in \R^n$, we can express its norm as:
\begin{equation}
    \|\vect{v}\| = \sqrt{\sum_{i=1}^{n} v_i^2}
    \label{eq:norm}
\end{equation}
